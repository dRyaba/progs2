# Progs2

Учебные задачи по программированию на C++ (НГУ).

## Задачи

### 1.1.2 — Класс String (инкапсуляция, конструкторы, перегрузка операторов)

Реализовать класс для работы со строками задаваемой длины.

**Функциональность:**
1. Инициализация: конструктор по умолчанию, конструктор от `char`, конструктор от `const char*`
2. Операторы: `+`, `+=` (конкатенация), `()` (подстрока), `*` (дубликация), `[]` (доступ по индексу), `-` (удаление подстроки)
3. Длина строки O(1)
4. Ввод/вывод в файл или на экран
5. Корректное управление памятью

**Демо:** замена всех вхождений str2 на str3 в str1. Вход: `input.txt` (три строки), выход: `output.txt`.

---

### 2.1.1 — Иерархия выражений (наследование, полиморфизм)

Реализовать абстрактный класс `Expression` и производные: `Number`, `Variable`, `Add`, `Sub`, `Mul`, `Div`.

**Функциональность:**
1. Создание выражения по строке (подвыражения в скобках)
2. Печать выражения
3. Символьное дифференцирование по переменной
4. Вычисление при заданном означивании переменных
5. Корректное управление памятью

**Демо:** прочитать выражение из `input.txt`, продифференцировать по "x", записать в `output.txt`.

---

### 3.2.1 — Шаблонный Vector (шаблоны)

Реализовать `template <typename T> class Vector`:
1. Добавление в конец
2. Удаление по индексу
3. Оператор `[]` для доступа
4. Циклический сдвиг влево/вправо
5. Печать элементов

**Демо:** выполнение команд из файла (ADD, REMOVE, PRINT, UPDATE, LSH, RSH). Вход: тип (I/D/S), число команд, команды. Выход: результаты PRINT и содержимое массива.

---

### 4.2.1 — Калькулятор выражений (STL)

Разбор и вычисление арифметического выражения с настраиваемыми приоритетом и ассоциативностью операций.

**Требования:**
- Свойства операций в `std::map`
- Разбор за O(N)
- Использовать `std::stack` и `std::queue`
- Обработка деления на ноль через исключения

**Вход:** 4 строки (свойства +, -, *, /) и выражение. **Выход:** результат или ERROR.

---

### 5.1.1 — Web Crawler (многопоточность)

Программа для перебора и скачивания страниц. Обязательный протокол: `file://`.

**Требования:**
1. Загрузка страницы на диск
2. Поиск ссылок `<a href="...">`
3. Рекурсивное сканирование
4. Вывод статистики: время работы, количество ссылок
5. Многопоточность с настраиваемым числом потоков

**Вход:** `<базовый_путь> <стартовая_страница> <количество_потоков>`. **Выход:** `<посещённых_страниц> <время_в_секундах>`.

---

## Сборка

```bash
cmake -B build
cmake --build build
```

## Запуск

Все программы читают `input.txt` и пишут `output.txt` в своей папке (или в текущей директории при запуске из `build/`):

- `build/task_01_string/task_01_string`
- `build/task_02_expression/task_02_expression`
- `build/task_03_vector/task_03_vector`
- `build/task_04_calculator/task_04_calculator`
- `build/task_05_crawler/task_05_crawler`

На Windows добавьте `.exe`.
